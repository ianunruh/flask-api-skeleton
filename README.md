## Requirements

* Python 3.5+

## Development

```bash
git clone git@github.com:ianunruh/flask-api-skeleton.git
cd flask-api-skeleton

pip install -U -r requirements.txt

./manage.py db upgrade
./manage.py runserver -rd
```
